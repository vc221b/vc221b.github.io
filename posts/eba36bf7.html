一、学习目标

能够说出React是什么
能够说出React的特点
能够掌握React的基本使用

二、React技术概述

1、概述

前端企业中主流：angular.js(过时)、vue(中国人，国内用多)、react(facebook)

https://zh-hans.reactjs.org/

作用：用于构建用户界面的JavaScript库，React主要用于构建Web应用

javascript库：jquery.js , bootstrap.js 原生javascript去开发

注册。register.js

设计组件，用户名和密码，单选按钮，用react组件->UI(user interface)->后台可以获取到这些信息

技术选型：听书app, 用什么技术来做

1）版本更新速度 2）有健全的文档与完善的社区 3）业务场景 4）现在team团队人会什么

2.特点

声明式：UI,html代码，所见即所得， react做渲染（将组件展现在页面）

基于组件：按钮，下拉菜单，标签页，导航栏（可复用）

学习一次，随处使用：react学习一次，ios，androd,abc,组件都是通用。

原来做手机app: ios，android->招两个开发人员

abc系统：abc系统

（java:一次编译class，到处运行linux,ios,windows）

三、环境安装

1 开发环境的工具必须统一（重点)

进企业->项目团队->项目服务器（git 版本管理)->OA（办公自动）

jdk11,tomat8,redis(nosql),软件，中间件，微服务springcloud springboot,不好排查

安装文档->按安装文档，用软件，中间件，什么样版本

保证你的环境和team是一致。

2.下载node.js

稳定，stable ,正式版

3.安装node.js

检查安装版本(>12)，安装是否成功

node -v

node在安装就已经给你配置好了path

java安装->配置环境变量path

4.npm

maven 包管理

后端：jdbc->mysql、oracle、sqlserver、db2->数据库驱动程序mysql5.1 mysql8驱动不一样

web->lib,把jar包放进去。

maven帮助我去维护项目中jar，想要那个我就从maven服务器，所有jar

npm对js库管理

前端：jquery.js、bootstrap.js、reactjs

npm服务器包含所有js插件，js库

npm install jquery.js

npm服务器搭建遥远，离我隔个太平洋。

阿里，新浪->cnpm器 从npm下js->国内开发cnpm

四、安装react应用

1.官网

https://reactjs.org/

2.新建目录，存入以后我们开发项目，不要放c

D:\androidProject

3.创建一个react应用

npx create-react-app hello3

如果不用管理员权限，permitted

4.运行

cd hello3

npm start

5.常用的端口

IP:Port

127.0.0.1 门牌号 中关村大街111号 乐乐酒店

tomcat 8080这个房间 ，port 酒店中一个房间（35600)->tomat->1000以下系统（6666，8888，9999）

mysql 3306-3309

oracle 1521->界面管理server->自动抢8080->4张光盘oracle9i 网络设置

sqlserver 1433

6.为什么可以react首页面

vscode->找到项目，使用文件夹打开

目录结构：

node_modules:项目库要用到的js库

src下面我们自己开发文件

index.js 入口文件 import App from ‘./App’; App.js

App.js 组件

在工具启动项目：终端-新建终端->npm start

package.json配置文件（不用管）
